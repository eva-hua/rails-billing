## Bill

This is a billing program developed with Rails 4.

#### Dependencies

  * mongodb  
  * Rails 4  


#### Usage

    git clone https://github.com/wusuopu/rails-billing billing
    cd billing
    bundle install
    vim config/mongoid.yml
    rails server
